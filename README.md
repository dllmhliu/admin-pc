# vue-admin

### 安装依赖包
```
npm install
```

### 运行
```
npm run serve
```

### 打包
```
npm run build
```

### 后台管理系统模板

#### 技术栈使用：
```
 vue-cli@3.0、vue-router、vuex、axios、element@2.82、echarts、@tinymce/tinymce-vue,tinymce、sass、eslint
```
